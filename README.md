API link estudiantes





Instalacion y ejecución
1.Clonar repositorio

2.Acceder a la carpeta
cd LinkStudents

3.Instalar dependencias
npm install

4.Configuración del entorno:
Copia el archivo .env.example y renómbralo a .env
cp .env.example .env

4.Inciar proyecto
    *Desarrollo
        npm run dev
    *Produccion
        npm start
